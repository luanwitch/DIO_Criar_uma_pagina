Estrutura Básica em HTML

(voltar)

O HTML (Hypertext Markup Language) é a linguagem padrão para a criação e estruturação de páginas web. Sua estrutura básica inclui as tags <html> para delimitar o início e o fim do documento, <head> para informações meta e de formatação, <title> para o título da página, <body> para o conteúdo visível e as tags de marcação como <h1> a <h6> para títulos, <p> para parágrafos, <a> para links, entre outras, para organizar e formatar o conteúdo da página.

Falando sobre Tags

(voltar)

As tags no HTML são elementos essenciais que estruturam e formatam o conteúdo de uma página web. Cada tag é delimitada por colchetes angulares e pode ter atributos para fornecer informações adicionais. Algumas tags comuns incluem <html> para o início/fim do documento, <head> para metadados, <title> para o título da página, <body> para o conteúdo visível, e tags como <h1>, <p>, e <a> para organizar e formatar textos, parágrafos e links. As tags HTML são cruciais para criar uma estrutura coerente e compreensível em páginas web.

Listas ordenadas e não ordenadas

(voltar)

Listas ordenadas <ol< são usadas para representar itens em uma sequência ordenada, marcados por números ou símbolos. Cada item é definido pela tag <li< Listas não ordenadas <ul< representam itens sem uma ordem específica, geralmente marcados por pontos ou símbolos. Ambos os tipos de listas melhoram a organização e legibilidade do conteúdo em páginas web, com a tag <li<indicando cada item na lista.
